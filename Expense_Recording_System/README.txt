Expense Tracker Application

This project provides a complete expense management system with:

A CLI (Command Line Interface) version

A GUI (Tkinter-based) version with charts and PDF reporting

FEATURES

Add, edit, and delete categories

Predefined and custom categories

Assign categories to expenses

Predictive category suggestions

Per-category budgets with alerts

Recurring expense support

CSV export (GUI)

PDF monthly report generation

Bar charts and pie charts

Data stored in JSON files

Expense_Tracker_GUI.py - GUI application

REQUIREMENTS

Python 3.8 or higher

Install additional libraries:
pip install matplotlib reportlab pillow

HOW TO RUN

CLI:
python expense_tracker.py

GUI:
python expense_tracker_gui.py

DATA STORAGE

Data is automatically saved in:
categories.json
expenses.json
